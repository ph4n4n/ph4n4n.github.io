Zero to Hero: HTML - Cấu trúc web cơ bản

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu chuẩn để tạo ra các trang web. Đây là nền tảng của mọi website hiện đại.

1. HTML là gì?

HTML là ngôn ngữ đánh dấu siêu văn bản, được sử dụng để tạo cấu trúc cho các trang web. HTML không phải là ngôn ngữ lập trình mà là ngôn ngữ đánh dấu.

Đặc điểm chính:

  • Markup Language: Sử dụng tags để đánh dấu nội dung
  • Structure: Tạo cấu trúc cho trang web
  • Semantic: Có ý nghĩa về mặt ngữ nghĩa
  • Accessible: Hỗ trợ accessibility
  • SEO-friendly: Tối ưu cho search engines

13. Best Practices

HTML Best Practices

  1. Sử dụng semantic HTML: <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  2. Cấu trúc heading đúng: H1 → H2 → H3 → H4 → H5 → H6
  3. Alt text cho images: Luôn có alt text mô tả
  4. Form accessibility: Label cho mọi input, proper validation
  5. Meta tags: Title, description, viewport
  6. Clean code: Indent đúng, comment khi cần
  7. Performance: Optimize images, minify HTML
  8. Cross-browser compatibility: Test trên nhiều browser
  9. Mobile-first: Responsive design
  10. SEO-friendly: Proper structure, meta tags

Common Mistakes to Avoid

HTML là nền tảng của web development. Việc nắm vững HTML semantic và best practices sẽ giúp bạn tạo ra những trang web chất lượng, accessible và SEO-friendly.
