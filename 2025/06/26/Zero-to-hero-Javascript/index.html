Zero to Hero: JavaScript - Từ cơ bản đến nâng cao

JavaScript là ngôn ngữ lập trình đa năng, từ frontend đến backend, từ web đến mobile. Đây là hướng dẫn toàn diện từ zero đến hero.

1. Giới thiệu và khái niệm cơ bản

JavaScript là gì?

JavaScript là ngôn ngữ lập trình động, được thiết kế để tạo ra các trang web tương tác. Ngày nay, JavaScript đã vượt ra khỏi browser và được sử dụng ở mọi nơi.

Đặc điểm chính:

  • Dynamic typing: Không cần khai báo kiểu dữ liệu
  • Interpreted: Chạy trực tiếp không cần compile
  • Object-oriented: Hỗ trợ OOP
  • Functional: Hỗ trợ functional programming
  • Event-driven: Xử lý sự kiện hiệu quả
  • Single-threaded: Một main thread với event loop
  • Prototype-based: Kế thừa dựa trên prototype

Use cases phổ biến:

  • Frontend: DOM manipulation, UI interactions
  • Backend: Node.js, Express.js, NestJS
  • Mobile: React Native, Ionic
  • Desktop: Electron, NW.js
  • IoT: Johnny-Five, Node-RED
  • Game Development: Phaser, Three.js
  • Data Visualization: D3.js, Chart.js

Lịch sử phát triển:

  • 1995: Brendan Eich tạo ra JavaScript tại Netscape
  • 1997: ECMAScript 1 được chuẩn hóa
  • 2009: ECMAScript 5 (ES5) - cải tiến lớn
  • 2015: ECMAScript 2015 (ES6) - modern JavaScript
  • 2016+: Hàng năm có phiên bản mới
