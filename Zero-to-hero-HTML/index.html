Zero to Hero: HTML - Cấu trúc web cơ bản

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu chuẩn để tạo ra các trang web. Đây là nền tảng của mọi website hiện đại.

1. Giới thiệu và khái niệm cơ bản

HTML là gì?

HTML là ngôn ngữ đánh dấu siêu văn bản, được sử dụng để tạo cấu trúc cho các trang web. HTML không phải là ngôn ngữ lập trình mà là ngôn ngữ đánh dấu.

Đặc điểm chính:

  • Markup Language: Sử dụng tags để đánh dấu nội dung
  • Structure: Tạo cấu trúc cho trang web
  • Semantic: Có ý nghĩa về mặt ngữ nghĩa
  • Accessible: Hỗ trợ accessibility
  • SEO-friendly: Tối ưu cho search engines
  • Cross-platform: Chạy trên mọi thiết bị
  • Standard: Tuân thủ chuẩn W3C
  • Extensible: Có thể mở rộng với CSS và JavaScript

Lịch sử phát triển:

  • 1991: HTML được tạo bởi Tim Berners-Lee
  • 1995: HTML 2.0 - Chuẩn đầu tiên
  • 1997: HTML 3.2 - Thêm tables và forms
  • 1999: HTML 4.01 - Chuẩn ổn định
  • 2000: XHTML 1.0 - HTML dựa trên XML
  • 2014: HTML5 - Chuẩn hiện đại

HTML vs XHTML vs HTML5

Feature HTML XHTML HTML5
Syntax Loose Strict Flexible
Case Case-insensitive Case-sensitive Case-insensitive
Quotes Optional Required Optional
Self-closing Not required Required Not required
Semantic Limited Limited Rich
Multimedia Limited Limited Native support
